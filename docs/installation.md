# Installation

1.  **Download** the add-on file (single `__init__.py` or `.zip` package).
2.  Open **Blender** -> **Edit** > **Preferences** > **Add-ons**.
3.  Click "**Install...**" and select the file.
4.  Search for "**FNA Adjuster**" and enable it.
5.  The panel appears in the **3D Viewport Sidebar** (N key) under the "**FNA Adjuster**" tab.
